Clase 1 - HTML: Estructura y Etiquetas Básicas

Este es un breve resumen de lo visto en la primera clase de Introducción a la Programación - Argentina Programa 4.0


¿Qué es un Navegador?

 Es un software que permite a un usuario el acceso a la web, interpretando las páginas (o la información de destintos tipos de archivos) y muestra su contenido de forma adecuada para que la entienda.

Entendiendo el FRONTEND

 La palabra "front" nos habla de algo que está "de frente", es decir, es la parte frontal de un sitio web, lo que ve y experimenta el usuario cuando ingresa e interactúa con él.

 Corresponde a todo el código que se ejecuta en el navegador de un usuario, e incluye el diseño, la estructura y los estilos de un sitio. Incluye el color, el fondo, animaciones, efectos, etc.

 Dentro del frontend podemos encontrar 3 tipos de leguaje: HTML, CSS y Javascript.

HTML: Un lenguaje que marca tendencia

 La sigla HTML significa:

 HT= Hipertextos. Hace referencia a que contiene páginas que se conectan entre sí y pueden navegarse a través de enlaces.

 ML= Lenguaje de Marcado. Es un conjunto de reglas de comunicación que utiliza etiquetas para definir el contenido y estructura de una página web, como título, párrafo, etc.

 Tiene una estructura básica que inicia con la declaración !Doctype, que indica el tipo de archivo y su versión (En HTML5 aparece como "!Doctype html"). Y luego le sigue el código html propiamente dicho.

 Este código tiene una sintaxis particular, compuesta por etiquetas. Estas tienen una estructura básica, con una apertura, un contenido y un cierre, aunque hay algunas excepciones donde no existe el cierre tradicional. Éstas a su vez, pueden tener contenido y atributos, y no pueden estar solapadas.

 De manera introductoria vamos a decir que las principales etiquetas en HTML son: html, head, title, body, h1 al h6, y p; pero hay muchas más.

CSS: Dale formato y color

 Es un lenguaje de estilo. Se encarga del formato y diseño visual.

Javascript: Sin movimiento es aburrido

 A diferencia de los otros dos, este es un lenguaje de programación, rápido y seguro, que utilizan la mayoría de los sitios web. Permite darle vidilla al sitio web, otorgando funciones e interacciones para el usiario. Se encarga de darle dinamismo.
