Clase 2 - HTML: Otras Etiquetas

En esta oportunidad te presentaremos otras etiquetas de gran utilidad en html y que te ayudarán a continuar aprendiendo en este curso de Introducción a la Programación - Argentina Programa 4.0


Etiquetas de separación

Nos permiten separar el texto entre párrafos intercalándolas entre ellos.

< hr > (horizontal rule)
Dibuja una línea horizontal y luego produce un salto de línea, separando así los párrafos.
< br > (line brake)
Produce un salto de línea en el texto (retorno de carro), como separador de párrafos.

Etiquetas de énfasis

Usadas para cambiar el aspecto del texto con el fin de destacarlo.

< strong >
Define el texto con fuerte importancia, usualmente se observa como "negritas".
< em >
Enfatiza el texto con el recurso de letras itálicas o cursivas.

Etiquetas de delineado

Coloca una línea horizontal sobre o debajo del texto.

< del > (deleted)
Produce una línea sobre el texto que encierra, dándole aspecto de "tachado".
< ins > (insert)
Muestra el texto entre etiquetas "subrayado", indicando que el mismo ha sido añadido al documento.

 Nota: con ambas etiquetas pueden usarse los siguientes atributos.

  • cite: para indicar una url a un documento que explique el motivo de la eliminación/inserción del texto.
  • datetime: para indicar el día y la hora en la que el texto fue eliminado,, insertado o modificado.
